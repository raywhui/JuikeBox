# JuikeBox: The Night. Remixed.

Ever wanted to enjoy a nice drink, but didn't know what to make with the random ingredients you have lying around? Look no further than Juikebox, the app designed to create drinks with the ingredients you have and to suggest music based on those ingredients. All you need are ingredients
